ID Card Replacement System

Student Interface Finance Department Interface ID Replacement Department Interface

Student Interface

Students can log in using their credentials and select the type of request: ID card renewal or ID card replacement. They can upload necessary documents and pay the fee online.

Finance Department Interface

The finance department can view pending requests, verify payment details, and approve or deny requests based on banking information provided by students.

ID Replacement Department Interface

The ID replacement department can see approved requests, check documents provided by students, and set appointments for ID pickup or deny requests.
